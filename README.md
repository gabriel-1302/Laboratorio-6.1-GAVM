# Hotel MONOLITH — Landing Page Estática

Landing page ultra-minimalista construida con **Astro** y **Tailwind CSS**, desplegada en **Amazon S3** con distribución CDN via **Amazon CloudFront**, usando **GitHub Actions** para CI/CD continuo.

## Stack

- **Framework:** Astro 4 (SSG)
- **Estilos:** Tailwind CSS
- **CI/CD:** GitHub Actions
- **Hosting:** Amazon S3 + CloudFront

## Desarrollo local

```bash
npm install
npm run dev
```
